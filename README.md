# AnchorToken (ANCH)

AnchorToken (ANCH) is a standard ERC-20 token deployed on the Polygon network.  
The contract is implemented using audited OpenZeppelin libraries and is designed for transparency, simplicity, and educational experimentation.

This repository contains the verified smart contract source code for AnchorToken.

---

## Token Overview

- **Token Name:** AnchorToken  
- **Symbol:** ANCH  
- **Network:** Polygon  
- **Standard:** ERC-20  
- **Decimals:** 18  
- **Total Supply:** Fixed at deployment  
- **Minting:** Disabled after deployment  
- **Blacklist / Whitelist:** None  
- **Transfer Restrictions:** None  

The token is fully transferable and does not include hidden control mechanisms.
